4. Or use Live Server in VS Code for easier testing.

## Deployment

This project can be deployed using GitHub Pages.

### GitHub Pages Steps
1. Push the project to GitHub.
2. Open the repository.
3. Go to **Settings**.
4. Click **Pages**.
5. Under **Source**, choose the `main` branch.
6. Set the folder to `/root`.
7. Save the changes.
8. Wait a few minutes for the live link.
